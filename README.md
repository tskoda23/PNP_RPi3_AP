# RPi3_AP
RPi 3 as hotspot


### Installation
`sudo chmod +x install.sh`
`sudo ./install.sh`

### Usage
#### Starting AP
`sudo start_ap <Your_SSID> <Your_PASSWORD>`

#### Stopping AP
`sudo stop_ap`

Original repository is from [PNPTutorials](https://github.com/PNPtutorials/PNP_RPi3_AP). I just added script that stops ap.
